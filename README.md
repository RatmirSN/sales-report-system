# Система отчетности по выполнению плана подключения новых абонентов

Автоматизированная система для формирования отчетов о выполнении плана по подключению новых абонентов сотрудниками отдела продаж.

**Версия без установки дополнительных приложений - работает в стандартном Excel**

## Функциональность

- **Доведенный план** - плановое количество подключений для каждого сотрудника
- **Факт** - фактическое количество подключений
- **Заявления в работе** - количество текущих заявок в обработке
- **Расчет выполнения плана** - процент выполнения и отклонение от плана
- **Статус выполнения** - автоматическая классификация (Выполнен/В процессе/Критично)
- **Условное форматирование** - цветовая индикация статуса

## Быстрый старт

### Вариант 1: Веб-приложение (рекомендуется для ввода данных)

1. Откройте файл `web_report.html` в браузере
2. Введите данные сотрудника через форму
3. Данные автоматически сохраняются в браузере
4. Отчет обновляется в реальном времени с расчетами

**Преимущества:**
- Сотрудники могут самостоятельно вносить данные
- Автоматический расчет выполнения плана
- Цветовая индикация статуса
- Сводная информация по всем сотрудникам
- Данные сохраняются между сессиями

### Вариант 2: HTML-файл для Excel

1. Откройте файл `sales_report.html` в Excel
2. Файл содержит готовый отчет с примером данных и форматированием
3. Редактируйте данные напрямую в Excel

### Вариант 3: CSV-файл

1. Запустите скрипт для создания CSV:
```powershell
powershell -ExecutionPolicy Bypass -File create_report.ps1
```
2. Откройте файл `sales_report.csv` в Excel
3. Добавьте формулы для расчетов (см. ниже)

## Добавление формул в Excel

После открытия CSV-файла добавьте следующие формулы:

### Выполнение % (колонка G)
Для строки 2:
```
=IF(D2=0,0,E2/D2*100)
```
Протяните формулу вниз для всех строк

### Отклонение (колонка H)
Для строки 2:
```
=E2-D2
```
Протяните формулу вниз для всех строк

### Статус (колонка I)
Для строки 2:
```
=IF(G2>=100,"Выполнен",IF(G2>=50,"В процессе","Критично"))
```
Протяните формулу вниз для всех строк

## Условное форматирование

Выделите колонку I (Статус) и добавьте условное форматирование:

1. **Зеленый** для "Выполнен"
   - Правило: Текст содержит "Выполнен"
   - Фон: светло-зеленый (#C6EFCE)
   - Текст: темно-зеленый (#006100)

2. **Желтый** для "В процессе"
   - Правило: Текст содержит "В процессе"
   - Фон: светло-желтый (#FFEB9C)
   - Текст: коричневый (#9C5700)

3. **Красный** для "Критично"
   - Правило: Текст содержит "Критично"
   - Фон: светло-красный (#FFC7CE)
   - Текст: темно-красный (#9C0006)

## Структура файла

| Колонка | Описание | Пример |
|---------|----------|--------|
| A | Сотрудник | Иванов И.И. |
| B | План | 50 |
| C | Факт | 52 |
| D | Заявок в работе | 3 |
| E | Выполнение % | 104.00 (формула) |
| F | Отклонение | 2 (формула) |
| G | Статус | Выполнен (формула) |

## Классификация статуса

- **Выполнен** - выполнение плана ≥ 100% (зеленый)
- **В процессе** - выполнение плана 50-99% (желтый)
- **Критично** - выполнение плана < 50% (красный)

## Веб-приложение

Файл `web_report.html` содержит полноценное веб-приложение с:

- **Форма ввода данных** - поля для ФИО, плана, факта и заявок в работе
- **Автоматические расчеты** - выполнение %, отклонение, статус
- **Цветовая индикация** - зеленый/желтый/красный в зависимости от выполнения
- **Сводная информация** - общие показатели по всем сотрудникам
- **Сохранение данных** - данные сохраняются в localStorage браузера
- **Удаление записей** - возможность удаления ошибочных записей
